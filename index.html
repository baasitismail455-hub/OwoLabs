The analysis follows three specific objectives: 1. Assess the serum levels of calcium, albumin, and phosphate in malaria-infected and non-infected children. 2. Compare biochemical parameters across malaria and control groups. 3. Determine correlations among parameters within malaria-infected children. All analyses were performed in **R**, and the steps are detailed below. --- # Data Cleaning and Preparation ```{r setup, message=FALSE, warning=FALSE} library(rio) # for data importation library(tidyverse) # for data wragling #importing the dataset owo <- import("C:/Users/user/Downloads/ola.xlsx") glimpse(owo) owo <- owo |> select(!`S/N`) # ro drop s/n # data cleaning owo <- owo |> filter(!(is.na(`SAMPLE NUMBER`) | `SAMPLE NUMBER` == "")) colnames(owo) <- c('sampleId', 'age', 'calcium', 'albumin', 'phosphatase') View(owo) # identify and inspect all unique labels in sampleId unique(owo$sampleId) # create a new variable (status) owo$status <- case_when( grepl('CONTROL', owo$sampleId, ignore.case = T) ~ 'control', #for control grepl('SAMPLE', owo$sampleId, ignore.case = T) ~ 'malaria', #for malaria TRUE ~ NA_character_ ) # remove std and na owo <- owo |> filter(!is.na(status)) glimpse(owo) #summary data summary(owo) table(owo$status) ``` # Descriptive and Comparative Statistics Descriptive statistics were generated to summarize calcium, albumin, and phosphate levels within each group. ``` {r} # statistical analysis #statistical summary. tapply(owo$calcium, owo$status, summary) tapply(owo$albumin, owo$status, summary) tapply(owo$phosphatase, owo$status, summary) tapply(owo$age, owo$status, summary) # compare parameters across the two status # using independent t-test calcium <- t.test(calcium ~ status, data = owo) # for calcium albumin <- t.test(albumin ~ status, data = owo) # for albumin phosphatase <- t.test(phosphatase ~ status, data = owo) # for phosphatase calcium albumin phosphatase ``` # Correlation Analysis by Malaria infected children only ``` {r} # objective three # correlate among biochemical parameters in malaria children # filter malaria children only malaria <- owo |> filter(status == 'malaria') View(malaria) # correlations cor_calcium_albumin <- cor.test(malaria$calcium, malaria$albumin) cor_calcium_phosphate <- cor.test(malaria$calcium, malaria$phosphatase) cor_albumin_phosphate <- cor.test(malaria$albumin, malaria$phosphatase) cor_calcium_albumin cor_calcium_phosphate cor_albumin_phosphate ``` #
